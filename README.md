## Project Overview

This repository contains my Digital Literacy Portfolio created as part of the CSE0001 course. The purpose of this project is to develop awareness about responsible technology usage, safe online communication, and professional digital presence.

Through this project, I explored various tools such as Canva, GitHub, LinkedIn, Google Forms, and coding practice platforms. Each task helped me understand how digital tools can be used effectively for academic and professional purposes.

---

## Tasks Included

### Task 1 – Digital Literacy Infographic  
A visual infographic explaining digital literacy concepts, safe internet practices, and responsible technology usage.

### Task 2 – Student Digital Portfolio  
Creation of professional accounts on platforms such as GitHub and LinkedIn to build an academic and career-ready digital identity.

### Task 3 – Coding & Collaboration Platforms  
Practice on coding platforms and development of a Digital Literacy Awareness Quiz using Google Forms.

### Task 4 – Professional Email & Etiquette  
Drafting formal emails and creating a checklist of proper social media behaviour.

### Task 5 – Cybercrime Awareness Case Study  
Research-based case study explaining cybercrime threats and prevention methods.
